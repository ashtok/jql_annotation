# JQL Annotation Pipeline

This repository contains the setup for the JQL (Judging Quality across Languages) annotation pipeline for multilingual data quality assessment.

## Setup

1. Create virtual environment: `uv venv --python 3.10`
2. Activate: `source .venv/bin/activate`
3. Install dependencies: See installation steps in docs
